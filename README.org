** Ejecución de los notebooks en su disco duro (recomendado)

Requisitos para ejecutar los notebooks localmente:

1. Instale Python:
   - https://www.python.org/downloads/
   - https://realpython.com/installing-python/

2. Instale el administrador de paquetes =pip=:
   - https://packaging.python.org/en/latest/tutorials/installing-packages/

3. Instale ~Jupyter Notebook~ o ~Jupyter Lab~ con =pip=:
   - https://jupyter.org/install
   - https://codigospython.com/descarga-e-instalacion-de-jupyter/

4. Instale el módulo [[https://pypi.org/project/nacal/][~nacal~]] con =pip=:
   - En el terminal: =pip install nacal=

5. Descargue el [[https://github.com/mbujosab/nacal-Jupyter-Notebooks][repositorio del curso]].

Luego, abra un terminal y navegue hasta la carpeta con los notebooks para ejecutar =jupyter notebook=. Esto abrirá una ventana en el navegador donde podrá manipular los cuadernos.
